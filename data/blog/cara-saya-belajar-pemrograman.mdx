Pada tahun ini Saya belajar banyak hal baru, seperti yang telah Saya tulis di artikel [Terima Kasih Tahun 2021](terima-kasih-tahun-2021).
Sekarang Saya ingin berbagi mengenai bagaimana cara saya belajar selama 1 tahun penuh ini.

Perlu diketahui bahwa metode yang Saya gunakan akan selalu berubah setiap tahunnya dikarenakan faktor waktu dan kondisi yang berubah.
Untuk tahun ini, Saya banyak melakukan pembelajaran melalui jaringan internet sehingga lebih mudah dan tidak menghabiskan banyak tenaga untuk pergi ke tempat belajar.

3 tahap umum bagi Saya dalam melakukan pembelajaran adalah:

## Cari ide terbaik

Ketahui apa yang Saya ingin pelajari dan cari ide sebanyak mungkin untuk menambahkan keyakinan
bahwa hal tersebut adalah hal yang Saya inginkan.

## Perbanyak Penelitian

Menyelam lebih dalam melalui jaringan internet, mencari informasi yang lebih banyak, dan
mengambil informasi yang lebih banyak. Ini akan sangat berguna untuk Saya sebelum melangkah
lebih jauh.

## Praktik Itu Penting

Lets do it! Praktikan apa yang telah Saya pelajari dan tidak lupa untuk mengikuti kegiatan
praktik seperti mengikuti workshop, mengikuti seminar, atau mengikuti kursus.

Melakukan apa yang Saya suka, jika Saya menyukai matematika maka kombinasikan pengetahuan mengenai matematika dengan keterampilan pemrograman. Ini akan membuat Saya lebih mudah dalam memahami dan mengimplementasikan kode.

```js:hash.js
n = []
p = []
for (a = 0; a < 3; a++) n[a] = fi() ? 1 : -1
step = ((Math.PI * 2) / size) * N
r = 0
for (f = 0; f < size; f++) {
  b = s[6](r * q[6] + s[3](r * q[3]) * h[5]) * n[0]
  a = 1 + b * h[0]
  d = s[7](r * q[7])
  e = -d
  d *= (2 - a) * n[1]
  e *= (2 - a) * n[2]
  c = (s[4](r * q[4] + s[5](r * q[5]) * h[7]) / 4) * h[6] * (a - (1 - h[0]))
  x = Math.sin(r * pr + c) * a + s[0][0](r * q[0]) * h[2] * d + s[1][0](r * q[1]) * h[3] * e
  y = Math.cos(r * pr + c) * a + s[0][1](r * q[0]) * h[2] * d + s[1][1](r * q[1]) * h[3] * e
  p[f] = [x * 110 + 200, y * 110 + 200]
  r += step
```
