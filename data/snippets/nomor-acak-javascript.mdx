Dalam JavaScript, kita dapat menghasilkan angka acak menggunakan fungsi `Math.random()`. Sayangnya, fungsi ini hanya menghasilkan angka floating-point antara 0 dan 1.
Dalam pengalaman saya, jauh lebih umum untuk membutuhkan bilangan bulat acak dalam rentang tertentu. Misalnya, nomor acak antara 10 dan 20.

Pembantu kecil ini memungkinkan kita melakukan itu!

```js:number.js
const random = (min, max) => Math.floor(Math.random() * (max - min)) + min;
```

## Penggunaan

```js:number.js
// Dapatkan nomor acak dari [10, 11, 12, 13]
random(10, 14);

// Dapatkan nomor acak dari 1 hingga 100 (inklusif)
random(1, 101);

// Dapatkan nomor acak dari -10 hingga 10 (inklusif)
random(-10, 11);
```
